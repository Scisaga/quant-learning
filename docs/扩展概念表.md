## 信息比率

信息比率（Information Ratio，简称 IR）是金融投资领域的一个绩效评估指标，主要用于衡量投资组合（如基金或策略）相对于基准指数（如沪深 300 或 S&P 500）的超额收益能力。它本质上是“每单位主动风险（也称为跟踪误差）所能获得的超额收益”的比率，帮助投资者评估基金经理的选股或择时能力是否值得承担额外风险。

### 核心含义
- **超额收益**：投资组合的回报率减去基准指数的回报率（例如，你的基金年化回报 15%，基准指数 10%，超额收益为 5%）。
- **主动风险（跟踪误差）**：投资组合回报与基准回报的偏差标准差，反映基金经理偏离基准的程度（越高，表示越“主动”管理，但风险也越大）。
- IR 值越高，表明基金经理在承担相同风险下，能产生更高的超额收益。通常，IR > 0.5 被视为优秀，IR > 1 被视为非常出色；负值表示表现不如基准。

### 计算公式
信息比率的计算公式为：
\[
IR = \frac{R_p - R_b}{\sigma_{TE}}
\]
- \( R_p \): 投资组合的回报率。
- \( R_b \): 基准指数的回报率。
- \( \sigma_{TE} \): 跟踪误差（Tracking Error），即 \( R_p - R_b \) 的标准差。

例如，如果一个基金的年化超额收益为 5%，跟踪误差为 4%，则 IR = 5% / 4% = 1.25，表示每单位风险获得 1.25 单位的超额收益。

### 与夏普比率（Sharpe Ratio）的区别
- **夏普比率**：衡量总回报相对于总风险（回报标准差）的效率，公式为 (回报 - 无风险利率) / 回报标准差。更适合评估绝对回报，不依赖基准。
- **信息比率**：更注重相对于基准的相对表现，强调“主动”管理能力。适合评估指数增强型基金或主动基金。
- 选择：如果你是绝对回报导向，用夏普比率；如果你跟踪指数（如被动投资），用信息比率。

### 实际应用
- **基金评估**：投资者用 IR 判断基金是否值得投资（高 IR 表示经理有“信息优势”）。
- **量化策略**：在 Qlib 等框架中，IR 用于回测策略绩效（如超基准的年化回报 / 风险）。
- **局限性**：IR 假设回报正态分布，且不考虑极端风险（如尾部风险）。在熊市中可能高估表现。

## CSI300

**CSI 300（沪深300）**是由中证指数有限公司编制的一个**综合性股票指数**，选取沪市和深市市值大、流动性好的 300 只股票组成，用于反映中国A股市场整体表现。

| 项目   | 说明                     |
| ---- | ---------------------- |
| 中文名称 | 沪深300指数                |
| 英文名称 | CSI 300 Index          |
| 指数代码 | **000300**             |
| 发布机构 | 中证指数有限公司（CSI）          |
| 样本范围 | 沪深两市市值与成交活跃度居前 300 只股票 |
| 性质   | 代表整个A股市场的“蓝筹指数”        |
| 常见用途 | 量化研究、指数基金、期货标的、回测样本集等  |

在 Qlib 中，`instruments="csi300"` 这一参数其实是一个**逻辑别名**，代表一组股票池，内部会映射到 沪深300 成分股的代码集合。


### `SH000300` 是什么

`SH000300` 是沪深300指数在 **上交所（Shanghai Stock Exchange）** 挂牌的标准证券代码：

* `SH` = 交易所前缀，表示上交所；
* `000300` = 指数代码；
* 合起来 `SH000300` 即“上证市场中的 000300 指数”；
* 同理，深交所的股票会以 `SZ` 开头，如 `SZ000001`。

在 Qlib 或金融数据接口中：

* 当你查询单只指数的行情（而非整个股票池）时，用 `SH000300`；
* 当你要获取沪深300 成分股的全集，用 `instruments="csi300"`。


## Alpha (α)：超额收益/非系统性收益
Alpha衡量的是投资组合在承担相应市场风险（Beta）后所获得的超越市场基准的额外回报。它代表了投资管理人的选股能力、时机把握能力或策略有效性。 

- 含义：Alpha是与市场波动无关的那部分收益，来源于深度研究、独特见解或数据分析优势，是主动投资管理追求的目标。
- 数值解释：
  - Alpha > 0：表示投资组合的表现优于预期基准收益，创造了正的超额价值。
  - Alpha = 0：表示投资组合的表现符合预期基准收益，没有创造额外价值。
  - Alpha < 0：表示投资组合的表现逊于预期基准收益。

- Alpha 策略：旨在寻找并利用市场无效性或定价错误，通过精选个股、利用量化模型等方法获取超额收益，是一种主动投资策略。

## Beta (β)：市场风险/系统性风险

Beta衡量的是投资组合或股票相对于整个市场（通常以某个指数为基准）的波动性或敏感性。它代表了市场平均收益，即随市场整体涨跌而产生的收益。 

- 含义：Beta反映了投资标的参与市场波动的程度，是无法通过分散投资来消除的系统性风险。
- 数值解释：
  - Beta = 1：表示该资产的波动幅度与市场整体一致。
  - Beta > 1：表示该资产的波动幅度大于市场，风险更高，潜在收益也更高（例如科技股）。
  - Beta < 1：表示该资产的波动幅度小于市场，风险较低，潜在收益也较低（例如公用事业股）。
  - Beta = 0：表示该资产与市场没有相关性（例如理论上的无风险资产，如短期国债）。
- Beta 策略：旨在捕获市场平均收益，通常通过购买指数基金或ETF来实现，是一种被动投资策略。
